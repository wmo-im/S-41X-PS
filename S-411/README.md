# S-411 – Sea Ice Information (S-100 Product Specification Support Repository)

## Overview

This repository supports development, testing, and implementation of **S-411 – Dynamic Ice Information**, an S-100 based product specification for digital sea ice data exchange and portrayal.

S-411 enables interoperable encoding of sea ice information for use in:
- ECDIS and S-100 compliant navigation systems
- Ice service operational environments
- Maritime safety and polar navigation applications

This repository is intended to support:
- Ice services
- Hydrographic offices
- S-100 developers
- Testbed participants
- Standards contributors

---

## Scope of This Repository

This repository may contain:

- Supporting documentation (reference PDFs)
- Development scripts (conversion utilities, object handling tools)
- Symbology references and portrayal examples
- Testing materials and draft updates
- Ice Object Catalogue related materials

⚠️ **Note:** Official normative versions of S-411 are maintained through the relevant governance process (IHO/WMO coordination). This repository does not replace formal publication channels.
